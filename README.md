# PHP_MVC_fromScratch

Create my own MVC structure in PHP.
Student project and solo work for a period of 14 days.

## Getting Started

All the needed files for have a visual example are in the repository.
You can get only the jQuery plugin file if you want to insert it in a block named with an id initialize to "game".
Takes a look at the main.js file to check how the plugin is called.

### Prerequisites

Phpmyadmin/Mysql or others like them. (& Apache Server or other)
Change the database username & password by your's in the "Database.php" class in the models directory.
